# HASS_NUC

Configuration for [Home Assistant](https://www.home-assistant.io/) running on an [Intel NUC](https://a.aliexpress.com/_EJ1VNbE) for a 5.5 room house, offering convenience automations over lights and climate while providing multiple intuitive user controls.

## Hardware specifications
* Intel Core 3 N355 
* 32GB DD5
* 2x ports LAN 2,5 GbE i226-v Intel 
* 1TB M.2 NVMe 2280
